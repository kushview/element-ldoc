Class el.DSPScriptPosition

Time information for DSP Scripts.

An object of this type is passed to to a DSP Script's process function conveying timing information.

Methods

DSPScriptPosition:recording () True if recording.
DSPScriptPosition:looping () True if looping
DSPScriptPosition:valid () True if this position is valid.
DSPScriptPosition:frame () Time in audio samples.
