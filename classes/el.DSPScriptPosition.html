Class el.DSPScriptPosition

Time information for DSP Scripts.

An object of this type is passed to to a DSP Script's process function conveying timing information.

Methods

el.dspscriptposition:playing () True if playing.
el.dspscriptposition:recording () True if recording.
el.dspscriptposition:looping () True if looping
el.dspscriptposition:valid () True if this position is valid.
el.dspscriptposition:frame () Time in audio samples.
el.dspscriptposition:seconds () Time in seconds.
