AudioBuffer:free ()
Free used memory. Invoke this to free the buffer when it is no longer needed. Once called, the buffer is no longer valid and WILL crash the interpreter if used after the fact. Only use this when you don't want to rely on the garbage collector to free memory.
AudioBuffer:isFloat ()
Returns true if audio data is 32bit float. Call this to find out audio data precision.
AudioBuffer:isDouble ()
Returns true if audio data is 64bit float. Call this to find out audio data precision.

AudioBuffer:applyGain (start, count, gain)
Applies a gain multiple to a region of all the channels. For speed, this doesn't check whether the sample numbers are in-range, so be careful!

Parameters:

  • start integer Sample index to start at
  • count integer Number of samples to process
  • gain number Amount of gain to apply
